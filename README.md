# Road-sign-detection-
This outlines the design, implementation, and performance of image processing pipeline which is designed to detect and classify road signs in both static images and video streams. It employs a combination of traditional computer vision techniques and heuristic approaches and avoids the use of deep learning methods to identify various types of road signs based on their colour, shape, and content. The project is structured to handle multiple input formats like single images, multiple images, and video frames in script format and provide detailed output for each detected sign in a structured format.
